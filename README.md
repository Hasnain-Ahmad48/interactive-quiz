Interactive Web Dev Quiz 

This is my Week 2 internship task. I built a fully responsive quiz application using vanilla HTML, CSS, and JavaScript to test basic knowledge of web development concepts.

Features

10 Questions: Covers HTML, CSS, and JavaScript basics.

Timer: You have 60 seconds to complete the quiz.
Instant Feedback: Shows if your answer was correct or wrong immediately.
Scoreboard: Displays your final score with a custom message at the end.
Responsive Design: Looks good on mobile and desktop.

Technologies Used
HTML5: For the structure of the app.
CSS3: For styling (Flexbox, Grid, and responsive layouts).
JavaScript: For the quiz logic, DOM manipulation, and timer functionality.
